Консольная игра "Виселица" на Java: угадывайте слова по буквам на русском языке. Ошибаетесь - и виселица будет достраиваться.

**Правила:**
* Слово загадывается игрой и скрыто звёздочками.
* В игре используются русские слова и русские буквы.
* Угадывать буквы необходимо по одной.
* Правильные буквы открываются в слове.
* За неправильные буквы добавляются элементы виселицы.
* Выводится список использованных букв.
* Уже использованные буквы не считаются за ошибку.
* Максимальное количество ошибок - 6.
* Игра завершается победой при полном открытии слова или поражением при достижении максимального количества ошибок.

*Дополнительно*: в IntelliJIdea игра запускается без проблем. Но чтобы запустить консольно - лично мне пришлось устанавливать Cmder и в нём вводить три команды:
``` 
chcp 65001
javac -encoding UTF-8 -d out src/main/java/org/learning/GallowsGame.java src/main/java/org/learning/GallowsPicture.java
java -Dfile.encoding=UTF-8 -cp out org.learning.GallowsGame
```

В стандартном cmd Windows и Windows PowerShell игра не запускалась даже с другими кодировками.
